认识我们

我们招聘的职位

快消品仓库出入库管理员/库管

面议

工作地点：南开区罗平道雅安光电创业园

任职要求：
1、男性，身体健康，40岁以下，有快消品仓库管理经验者优先录取；
2、踏实肯干，耐心细致，有责任心有吃苦耐劳精神；
3、会开车，C1驾照者优先录取；
4、退伍军人，共产党员，应届毕业生优先录取；
5、可接受实习生和培训生
任职资格：
1.入库清点、按库位入库；
2.出库拣货、按清单确保货物与订单一致；
3、确认、打印销售、出库订单；
4、整理货位、关注货物日期，保证先入先出，临期商品预警。

面议

4、负责所辖区域的卫生清洁，引导顾客遵守店铺公共秩序；
